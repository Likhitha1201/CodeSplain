# CodeSplain

## Project Overview

CodeSplain is a web-based application built using **React and JavaScript** that provides a development environment for working with code. The project uses a modern full-stack setup with a React frontend and an Express backend. It integrates tools for code editing, testing, and debugging to help developers understand and work with code more efficiently.

## Tech Stack

**Frontend**

* React 18
* JavaScript
* React Router
* Tailwind CSS
* Monaco Editor

**Backend**

* Node.js
* Express.js
* Prisma ORM

**Testing & Debugging**

* Jest
* React Testing Library
* Playwright
* Browser Debugger

**Other Tools**

* Axios
* SWR
* Classnames
* Dotenv
* Nodemon
* Concurrently

## Features

* Interactive code editor using Monaco Editor
* Component-based React architecture
* API communication using Axios
* Backend server with Express
* Database integration using Prisma
* Client and server running concurrently
* Unit and UI testing support using Jest and Playwright

## Installation

Clone the repository:

```
git clone https://github.com/Likhitha1201/CodeSplain.git
```

Navigate to the project folder:

```
cd CodeSplain
```

Install dependencies:

```
npm install
```

## Running the Application

Start both **client and server**:

```
npm start
```

Client runs on:

```
http://localhost:3000
```

Server runs on:

```
http://localhost:8000
```

## Running Tests

Run tests using Jest:

```
npm test
```

Debug tests:

```
npm run test:debug
```

## Build for Production

```
npm run build
```
